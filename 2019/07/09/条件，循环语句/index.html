条件，循环语句

mat's blog
2019-07-09
  1. assert断言用法：
    ···python
    if not condition:
    raise AssertionError()
    ···
    如果不满足assert后面的条件，就会抛出异常
  2. 三元运算符：
    a if b else c–如果满足b，返回a, 否则返回c
  3. zip函数返回的是一个包含元组的列表：
    name = [“ann”, “beth”]
    age = [12,15]
    zip(name,age)
    [(“ann”,12),(“beth”,15)]
    for name, age in zup(name,age);

注意zip可以压缩不同长度的序列，按照最短的来。

  2. 字典，对象，注意事项：
    x = {“name”:”matian”, “year”:18}
    y = x
    x = {}
    y = {}
    初始字典就会“漂”在内存中，python解释器就会删除这个字典（垃圾收集）
    del x 删除的是一个对象的引用，以及这个名字本身（x），但是并没有删除这个对象，（x对应的值）
